Tile color compares each panel to its string median — answers "is this panel keeping up with neighbors right now?". Low output at dawn, dusk, or under cloud cover is expected and shifts the median, not the color. The small stripe at the bottom of a tile compares to the per-panel nameplate rating (when set) using the same green/yellow/red bands — useful for spotting cases where the whole string is below rating but individual panels still look "on par" with each other.
